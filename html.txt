1.what is Html?
html stands for hypertext markup language. it is used to define the structure of the web page using diffrernt 
elements and tags.

2.tags
<!Doctype> : it declears the type of document in html
<html>   : it is the root element
<body>   : it contains the main content of the web page
<link>   : links the internal and external links in the web page
<title>  : title of the web page
<head>   : contains metadata(it represents seo releated data)
metadata : seo releated description
<h1> to <h6> : heading tags (size of text decreaes or increases according to tag <h1>:: biggest tag <h6>:: smallest tag)
<p>      : paragraph tag 
<b>      : it makes text bold
<br>     : it breaks the line between two sentence.(##it doesnot have closing tag) (empty tag)
<hr>     : it creates a Horizontal line (##it doesnot have closing tag) (empty tag)
<strong> : it makes text bold (used in important text)
<i>    : it makes text italic
<em>   : it makes text italic(used in important text)
<mark> : Highlighted text
<del>  : Deleted text
<small> : smaller text
<sub>  : Subscript (eg: Hâ‚‚O)

<ul>   : Unordered list
<li>   : listed items
<table>: table container

<td>   : table data
